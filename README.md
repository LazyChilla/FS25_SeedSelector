# FS25_SeedSelector

**Version 1.0.0.0 · Autor: LazyChilla · Multiplayer: ✅**

## 🇩🇪 Deutsch

### Was macht der Mod?
Seed Selector ersetzt das Standard-Durchklicken der Saatgutsorten (Y-Taste) durch eine übersichtliche, scrollbare Liste. Die Liste öffnet sich direkt am Bildschirm, zeigt alle Früchte der aktuellen Sämaschine alphabetisch sortiert und zeigt säbare Früchte oben an. Für jede Frucht ist der Saatzeitraum aus dem Anbaukalender sichtbar.

### Bedienung
| Aktion | Taste |
|--------|-------|
| Liste öffnen/schließen | **Y** (an der Sämaschine) |
| Tastatur-Fallback | **Alt rechts + F** |
| Scrollen | Mausrad |
| Frucht wählen | Linksklick |
| Abbrechen | Klick außerhalb der Liste / ESC |

### Anzeige
- **Gelb** = aktuell gewählte Frucht
- **Grün** = säbar in dieser Jahreszeit, mit Saatfenster (z.B. `Mär–Jun`)
- **Grau / ausgegraut** = nicht in Saison, mit Saatfenster
- **ganzjährig** = keine Saisonbeschränkung

### Installation
1. `FS25_SeedSelector.zip` in den Mods-Ordner kopieren
2. Mod im Spiel aktivieren
3. Sämaschine ankuppeln → Y drücken

### ⚠️ Kompatibilität / Bekannte Konflikte
**NICHT zusammen mit `FS25_SeedSelect` (Seed Select von plutogaming91) verwenden!**
Beide Mods ersetzen die Y-Taste an der Sämaschine. Bei gleichzeitiger Aktivierung kommt es zu unvorhersehbarem Verhalten. Nur einen der beiden Mods aktivieren.

### Multiplayer
Funktioniert im MP. Host und Clients können die Liste unabhängig voneinander öffnen. Die tatsächliche Fruchtwahl wird über das normale Spiel-Netzwerk synchronisiert.

### Bekannte Einschränkungen
- Der MP-Test (Host ↔ Client Scrollen) steht noch aus – bei Problemen bitte im GitHub-Issue melden.
- Icon-Konvertierung mit dem Giants Texture Tool wird nachgereicht.

## 🇬🇧 English

### What does this mod do?
Seed Selector replaces the default seed cycling (Y key) with a clear, scrollable list. The list opens on screen, shows all crops of the current sowing machine sorted alphabetically, and puts plantable crops at the top. The sowing season from the farming calendar is displayed for each crop.

### Controls
| Action | Key |
|--------|-----|
| Open / close list | **Y** (while in sowing machine) |
| Keyboard fallback | **Right Alt + F** |
| Scroll | Mouse wheel |
| Select crop | Left click |
| Cancel | Click outside the list / ESC |

### Display
- **Yellow** = currently selected crop
- **Green** = plantable this season, with sowing window (e.g. `Mar–Jun`)
- **Grey / dimmed** = not in season, with sowing window
- **all year** = no season restriction

### Installation
1. Copy `FS25_SeedSelector.zip` into your mods folder
2. Activate the mod in-game
3. Attach a sowing machine → press Y

### ⚠️ Compatibility / Known Conflicts
**Do NOT use together with `FS25_SeedSelect` (Seed Select by plutogaming91)!**
Both mods replace the Y-key action on sowing machines. Running both simultaneously causes unpredictable behaviour. Activate only one of the two mods.

### Multiplayer
Works in MP. Host and clients can open the list independently. The actual crop selection is synchronised through the normal game network.

### Known limitations
- Full MP test (host ↔ client scrolling) is still pending – please report issues on GitHub.

## 🇫🇷 Français

### Qu'est-ce que fait ce mod ?
Seed Selector remplace le changement de semence par défaut (touche Y) par une liste claire et défilante. La liste s'ouvre à l'écran, affiche toutes les cultures du semoir en cours par ordre alphabétique et place les cultures plantables en haut. La fenêtre de semis du calendrier agricole est affichée pour chaque culture.

### Commandes
| Action | Touche |
|--------|--------|
| Ouvrir / fermer la liste | **Y** (dans le semoir) |
| Raccourci clavier alternatif | **Alt droit + F** |
| Faire défiler | Molette de la souris |
| Sélectionner une culture | Clic gauche |
| Annuler | Clic en dehors / ESC |

### Affichage
- **Jaune** = culture actuellement sélectionnée
- **Vert** = plantable cette saison, avec fenêtre de semis (ex. `Mar–Jun`)
- **Gris / estompé** = hors saison, avec fenêtre de semis
- **toute l'année** = aucune restriction saisonnière

### Installation
1. Copier `FS25_SeedSelector.zip` dans le dossier des mods
2. Activer le mod en jeu
3. Attacher un semoir → appuyer sur Y

### ⚠️ Compatibilité / Conflits connus
**N'utilisez PAS avec `FS25_SeedSelect` (Seed Select de plutogaming91) !**
Les deux mods remplacent l'action de la touche Y sur les semoirs. Les utiliser simultanément entraîne un comportement imprévisible. N'activez qu'un seul des deux mods.

### Multijoueur
Fonctionne en mode multijoueur. L'hôte et les clients peuvent ouvrir la liste indépendamment. La sélection de culture est synchronisée via le réseau normal du jeu.

## 🇮🇹 Italiano

### Cosa fa questa mod?
Seed Selector sostituisce il ciclo di sementi predefinito (tasto Y) con un elenco chiaro e scorrevole. L'elenco si apre sullo schermo, mostra tutte le colture della seminatrice corrente in ordine alfabetico e mette le colture piantabili in cima. La finestra di semina dal calendario agricolo è visualizzata per ogni coltura.

|--------|-------|

| Scorciatoia tastiera | **Alt destro + F** |
| Scorrere | Rotella del mouse |
| Selezionare una coltura | Clic sinistro |
| Annullare | Clic fuori dalla lista / ESC |

### Visualizzazione
- **Giallo** = coltura attualmente selezionata
- **Verde** = piantabile questa stagione, con finestra di semina (es. `Mar–Giu`)
- **Grigio / opaco** = fuori stagione, con finestra di semina
- **tutto l'anno** = nessuna restrizione stagionale

### Installazione
1. Copiare `FS25_SeedSelector.zip` nella cartella mod
2. Attivare la mod nel gioco
3. Collegare una seminatrice → premere Y

### ⚠️ Compatibilità / Conflitti noti
**NON usare insieme a `FS25_SeedSelect` (Seed Select di plutogaming91)!**
Entrambe le mod sostituiscono l'azione del tasto Y sulle seminatrici. L'uso simultaneo provoca comportamenti imprevedibili. Attivare solo una delle due mod.

### Multigiocatore
Funziona in modalità multigiocatore. Host e client possono aprire la lista in modo indipendente. La selezione della coltura viene sincronizzata tramite la rete normale del gioco.

## 🇵🇹 Português

### O que faz este mod?
O Seed Selector substitui o ciclo de sementes padrão (tecla Y) por uma lista clara e rolável. A lista abre no ecrã, mostra todas as culturas da semeadora atual em ordem alfabética e coloca as culturas plantáveis no topo. A janela de sementeira do calendário agrícola é apresentada para cada cultura.

### Controlos
| Ação | Tecla |
|------|-------|
| Abrir / fechar lista | **Y** (na semeadora) |
| Atalho de teclado | **Alt direito + F** |
| Rolar | Roda do rato |
| Selecionar cultura | Clique esquerdo |
| Cancelar | Clique fora da lista / ESC |

### Visualização
- **Amarelo** = cultura atualmente selecionada
- **Verde** = plantável nesta época, com janela de sementeira (ex. `Mar–Jun`)
- **Cinzento / esbatido** = fora de época, com janela de sementeira
- **o ano todo** = sem restrição sazonal

### Instalação
1. Copiar `FS25_SeedSelector.zip` para a pasta de mods
2. Ativar o mod no jogo
3. Ligar uma semeadora → pressionar Y

### ⚠️ Compatibilidade / Conflitos conhecidos
**NÃO usar com `FS25_SeedSelect` (Seed Select de plutogaming91)!**
Ambos os mods substituem a ação da tecla Y nas semeadoras. O uso simultâneo causa comportamento imprevisível. Ativar apenas um dos dois mods.

### Multijogador
Funciona em multijogador. Host e clientes podem abrir a lista de forma independente. A seleção de cultura é sincronizada através da rede normal do jogo.

## 🇪🇸 Español

### ¿Qué hace este mod?
Seed Selector reemplaza el ciclo de semillas predeterminado (tecla Y) por una lista clara y desplazable. La lista se abre en pantalla, muestra todos los cultivos de la sembradora actual en orden alfabético y coloca los cultivos plantables en la parte superior. La ventana de siembra del calendario agrícola se muestra para cada cultivo.

### Controles
| Acción | Tecla |
|--------|-------|
| Abrir / cerrar lista | **Y** (en la sembradora) |
| Atajo de teclado | **Alt derecho + F** |
| Desplazar | Rueda del ratón |
| Seleccionar cultivo | Clic izquierdo |
| Cancelar | Clic fuera de la lista / ESC |

### Visualización
- **Amarillo** = cultivo actualmente seleccionado
- **Verde** = plantable en esta temporada, con ventana de siembra (ej. `Mar–Jun`)
- **Gris / atenuado** = fuera de temporada, con ventana de siembra
- **todo el año** = sin restricción estacional

### Instalación
1. Copiar `FS25_SeedSelector.zip` en la carpeta de mods
2. Activar el mod en el juego
3. Acoplar una sembradora → pulsar Y

### ⚠️ Compatibilidad / Conflictos conocidos
**¡NO usar junto con `FS25_SeedSelect` (Seed Select de plutogaming91)!**
Ambos mods reemplazan la acción de la tecla Y en las sembradoras. Usarlos simultáneamente provoca un comportamiento impredecible. Activa solo uno de los dos mods.

### Multijugador
Funciona en modo multijugador. El host y los clientes pueden abrir la lista de forma independiente. La selección de cultivo se sincroniza a través de la red normal del juego.
